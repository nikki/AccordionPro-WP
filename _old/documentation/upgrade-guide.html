Accordion Pro WP

A Responsive Accordion Plugin for WordPress from Stitch UI

Upgrading to v2.1

Before we get started, let's backup your accordions. Better safe than sorry.

Backup your Accordions

To backup your accordions, go to the WordPress admin panel, and in the sidebar click Tools > Export. On the Export screen you'll see an option that says 'Choose what to export' - select 'Accordions' and click 'Download Export File'. Put this file somewhere for safe keeping - you may need it at a later date. Here's hoping you don't!

Deactivate v2.0

Head over to the Plugins section of your WordPress admin panel, and click the 'Deactivate' link underneath the entry for Accordion Pro WP. The plugin is now inactive.

Open your FTP program of choice, and delete the old accordionpro_wp folder from your WordPress plugins directory. Upload the new accordionpro_wp folder in place.

Do not click the Delete link in the plugins panel. That would delete the plugin files and all of the plugin data (your accordions).

All Done!

That's everything! Enjoy the update!

Importing Accordions from Backup

In the highly unlikely event that you need to restore your accordions from backups - (did you accidentally click the 'Delete' link?) - you'll need to first install the WordPress Importer plugin. Once you've installed and activated the plugin, go to Tools > Import > WordPress, click 'Choose file', select your XML backup file, and click 'Upload file and import'.

On the next screen, 'Assign Authors', use the dropdown box to assign the accordions to an existing user. Hit 'submit', then go back to the Accordion Pro management panel. All your old accordions will have been restored.

Remember to remove the WordPress Importer plugin once you're finished.
