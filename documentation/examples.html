Note: when the linkable option is set, you can link to slides by using #yourDivId-slide-1, #yourDivId-slide-2, etc. in the link's href attribute.

  • Links:
  • #one-slide-1
  • #one-slide-2
  • #one-slide-3
  • #one-slide-4
  • #one-slide-5

Options used:

{
    theme : 'stitch',
    colour : {
        scheme : 'blue',
    },
    tab : {
        icon : 'chevron',
        font : 'Arial'
    },
    linkable : true
}
